# Repository Summarizer Prompt Strategies

## Suggested prompts
- Function-level: "Read `backend/src/services/inventoryService.js` and summarize its purpose, entry point, dependencies, and output shape."
- Folder-level: "Survey `frontend/src/components` and summarize how the files collaborate, which file feels like the entry point, and which files a newcomer should read first."

## What to record
- Which strategy produced the clearest answer.
- How much code context you had to pass.
- Which answer was easier to turn into README prose.
